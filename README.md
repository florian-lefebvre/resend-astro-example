# Resend with Astro

This examples shows how to use Resend with [Astro](https://astro.build/).

## Instructions

1. Define environment variables in .env file.
2. Install dependencies:

```
pnpm install
```

3. Run Astro locally:

```
pnpm dev
```

4. Open URL in the browser:

```
http://localhost:4321/api/send
```
